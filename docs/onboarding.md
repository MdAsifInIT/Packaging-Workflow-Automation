# Onboarding

## Prerequisites
- Git
- Jenkins admin access (or CI runner)
- n8n cloud account and credentials
- OpenAI API key (store in n8n, do not commit)

## Steps
1. Clone repo.
2. Import `templates/n8n-workflows/intake-openai.json` into n8n cloud.
3. Review `templates/psadt-template` and `examples/sample-app`.
4. Configure Jenkins credentials and create a Windows agent.
